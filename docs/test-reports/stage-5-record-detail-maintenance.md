# 阶段 5 测试报告：行为详情与记录维护

## 范围

本次验证覆盖行为详情页、历史记录列表、备注编辑、删除二次确认、称重数值输入，以及首页/日历数据联动的基础路径。

## 已实现内容

- 首页行为卡可进入行为详情页。
- 日历记录可进入对应行为详情页。
- 行为详情页展示频率、近 30/90/180 天次数、累计记录数和历史记录。
- 历史记录支持备注编辑。
- 历史记录支持删除，删除前二次确认。
- 称重行为打卡支持输入体重，保存 `value` 和 `valueUnit = kg`。
- 日历记录和详情记录摘要展示时间、数值和备注。

## 待手工验证

- 首页点击行为“详情”后是否进入正确行为详情页。
- 日历记录点击“详情”后是否进入正确行为详情页。
