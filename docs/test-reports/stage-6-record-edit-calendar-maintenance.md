# 阶段 6 测试报告：记录编辑、日历维护与联动验证

## 范围

本阶段验证日历补录称重数值、记录日期/时间编辑、日历记录直接编辑/删除、详情页底部导航优先级修复、首页/日历/详情页数据联动，以及 Debug 构建和模拟器启动。

## 已实现内容

- 日历补录支持称重行为输入体重数值，并保存 `value` 与 `valueUnit = kg`。
- 称重补录对空值、非法值、非正数进行拦截。
- 记录支持编辑日期、时间、备注。
- 称重记录支持编辑体重数值。
- 日历记录列表支持直接编辑和删除。
- 删除记录前有二次确认。
- 行为详情页历史记录支持编辑和删除。
- 详情页打开状态下点击底部导航，会关闭详情页并切换到目标 Tab。
- 首页、日历、详情页在新增、编辑、删除后同步刷新。

## 风险

- 当前记录编辑依赖手工输入 `yyyy-MM-dd` 和 `HH:mm`，需要重点验证格式错误提示。
- 删除入口同时存在于日历和详情页，需要手工验证二次确认一致性。
- 自动化测试仍不足，建议后续补 UI 测试覆盖称重校验、跨日期编辑和删除联动。
