# CatRe 需求状态

更新日期：2026-05-22

## 当前进度

当前已完成阶段 0-9。项目已具备 Android 本地应用、Room 数据存储、首次创建猫、默认行为、首页打卡统计、行为管理、频率规则基础、日历视图、补录、行为详情、记录编辑删除、称重数值输入、记录紧凑只读展示、记录三点菜单编辑删除、行为管理清单优先入口、中文图标选项、色块颜色选择、多猫管理基础闭环、行为删除、头像 URI 和日历二级补录能力。

## 已完成需求

### 阶段 0：工程基础

- Android 工程已创建。
- Kotlin + Jetpack Compose 基础界面已接入。
- Gradle Wrapper 已配置。
- 应用可构建 Debug APK。

### 阶段 1：基础数据

- 首次启动可创建第一只猫。
- 默认行为写入本地数据库。
- Room 数据库、Entity、DAO、Repository 基础结构已接入。
- 首页可读取当前猫咪与行为数据。

### 阶段 2：首页打卡与统计

- 首页行为卡支持一键打卡。
- 打卡记录写入 `check_in_records`。
- 首页展示上次打卡日期、距离上次天数、近 30/90/180 天次数。
- 今日状态按设备本地日期计算。
- 首页汇总展示行为数量和累计记录数。

### 阶段 3：行为管理与频率基础

- 底部导航已接入：首页、日历、行为、我的。
- 底部导航固定在屏幕底部。
- 行为管理页支持查看、新增和编辑行为。
- 行为支持名称、图标 key、颜色、首页展示开关和频率规则。
- 频率规则支持按需记录、每 N 天、每周 N 次、每月 N 次。
- 首页按 `showOnHome` 展示行为。
- 首页展示频率状态。
- 首页接入 CatRe logo 和背景图。

### 阶段 4：日历与补录基础

- 日历页支持当前月份月历展示。
- 支持上月、下月切换。
- 日期格展示当天记录数量。
- 点击日期后展示当天记录列表。
- 支持为选中日期补录行为记录。
- 补录后首页统计和日历记录数通过 Flow 联动刷新。
- 底部导航由字母改为图标，并有选中态区分。

### 阶段 5：行为详情与记录维护

- 首页行为卡支持进入行为详情页。
- 日历记录支持进入对应行为详情页。
- 行为详情页展示频率、近 30/90/180 天次数、累计记录数和历史记录。
- 历史记录支持备注编辑。
- 历史记录支持删除，删除前二次确认。
- 首页称重行为打卡支持输入体重数值。
- 称重记录保存 `value` 和 `valueUnit = kg`。
- 日历记录和详情记录摘要展示时间、数值和备注。

### 阶段 6：记录编辑与统一记录展示

- 日历补录支持称重数值输入。
- 记录支持编辑日期和时间。
- 日历记录列表支持直接编辑记录。
- 日历记录列表支持直接删除记录，删除前二次确认。
- 记录摘要统一展示时间、数值、单位和备注。
- 编辑记录时间后，记录会移动到对应日期。
- 修复行为详情页打开时点击底部导航无法切换的问题。
- 点击底部导航会关闭详情页，未保存修改不保留。

### 阶段 7：记录展示与行为管理体验优化

- 行为详情页历史记录默认只读紧凑展示，不再直接展开编辑表单。
- 日历当天记录列表默认只读紧凑展示，不再直接展开编辑表单。
- 记录项右上角支持三点菜单，提供编辑和删除操作。
- 点击编辑后才展开记录编辑表单。
- 删除记录继续保留二次确认。
- 行为管理默认进入行为清单。
- 行为管理右上角 `+` 按钮进入新增行为。
- 新增行为保存或取消后返回行为清单。
- 行为编辑仍从行为条目进入。
- 行为新增和编辑不再向用户展示图标 key，改为中文图标选项。
- 行为新增和编辑不再向用户展示颜色代码，改为色块和中文颜色名选择。

### 阶段 8：多猫管理闭环

- “我的”页支持查看所有猫咪资料。
- 支持新增多只猫咪，字段包含昵称、性别、生日、品种和备注。
- 支持切换当前猫咪，当前猫咪在列表中有明确标识。
- 当前猫咪选择会持久化，应用重启后可恢复。
- 支持编辑猫咪基础资料。
- 支持删除猫咪二次确认。
- 删除猫咪时同步删除该猫咪关联记录。
- 删除当前猫咪后自动回退到剩余猫咪；删除最后一只猫咪后进入无猫状态。
- 首页、日历、行为详情和记录列表按当前猫咪隔离展示。
- 新增打卡和日历补录自动绑定当前猫咪。
- 无当前猫咪时阻止补录并提示先新增猫咪。

### 阶段 9：行为删除、猫咪头像与日历补录优化

- 行为编辑页支持删除行为。
- 删除行为前弹出二次确认，并说明关联记录会一并删除。
- 删除行为时同步删除该行为关联记录。
- 行为删除使用 Room `withTransaction` 包裹关联记录清理和行为删除。
- 猫咪编辑页支持填写头像 URI。
- 首页、猫咪列表和猫咪编辑页支持展示头像，头像为空或加载失败时显示占位头像。
- 日历页补录入口改为“补录”按钮。
- 点击“补录”进入二级补录表单。
- 补录表单支持日期、行为、体重和备注输入。
- 补录保存后返回日历并刷新记录，取消后返回日历且不保存。
